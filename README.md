## Informações básicas sobre a criação da linguagem
  - **Criador:** James Gosling
  - **Empresa responsável:** Sun Microsystem, comprada pela Oracle em 2008
  - **Data de lançamento:** 23 de maio de 1995
  - **Primeiro nome:** *Oak*
